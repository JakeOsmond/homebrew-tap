# Homebrew Tap

Custom Homebrew formulae by JakeOsmond.

## Install

```bash
brew tap JakeOsmond/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `bmad-quick-setup` | One command to create a BMAD project, install it, and open Claude |

### bmad-quick-setup

```bash
brew install bmad-quick-setup
```

Then just run:

```bash
bmad my-project-name
```
